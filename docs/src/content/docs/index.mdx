## Sobre o projeto

O **Chronos** é o backend deste repositório: uma aplicação [FastAPI](https://fastapi.tiangolo.com/) com OpenAPI interativa em `/api/v1/docs` quando o servidor está em execução. Os modelos usam [Pydantic](https://docs.pydantic.dev/); a camada de dados usa [SQLAlchemy](https://www.sqlalchemy.org/) com [PostgreSQL](https://www.postgresql.org/) via [psycopg](https://www.psycopg.org/).
